Dino Finder

DinoFinder is an object-orientated PHP project, styled with SASS, that fetches and displays information about different dinosaurs, from a MySQL database.
I worked on this project with a team of people working in Scrum, my main roles involved working on implementing Hydrator classes and their associated SQL queries as well as assisting with Unit Tests and contributing to design decisions.

Github Live Release

Word Nerd

Word Nerd is a JavaScript web application (based on Wordle) that encourages users to guess a five-letter word, within six guesses, providing feedback along the way. The project includes a words.json file that can be customised to add or edit five-letter words of your choice.
I worked on this project with a team of people working in Scrum, in this project I created a function that dynamically generated the tiles and assigned them css classes to represent whether or not they were correct letter guesses. I also worked on the event listener that handled how the game ran including a guess counter and validation/sanitisation on the text entry.

Github Live Release

Job Listing Website

This mock Job Listing Website was a team project that we developed in React.js using SCSS stylesheets to work from and implement an existing design and wireframe.
In this project we used an API that was provided to us to query a DB of jobs and assorted information about them. The features I helped develop for this project was the search functionality that utilised an API fetch as well as triggering a conditional render that decided what information was displayed to the user on the front end.
